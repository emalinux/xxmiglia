# Agriturismo Ciapéleta – Sito Web

Sito web ufficiale dell’**Agriturismo Ciapéleta**, struttura ricettiva immersa nella natura, realizzato come **sito statico moderno**, veloce e ottimizzato per i motori di ricerca.

🌐 **Sito online:** https://agriturismociapeleta.net/

---

## 🎯 Obiettivi del progetto

- Presenza web chiara e professionale
- Massima velocità di caricamento
- Struttura semplice e facilmente manutenibile
- Ottimizzazione SEO tecnica e contenutistica
- Nessuna dipendenza da CMS pesanti o plugin superflui

---

## 🛠️ Stack tecnologico

- **Generatore statico:** Hugo  
- **Markup:** HTML5  
- **Stili:** CSS personalizzato  
- **SEO:** metadata, struttura semantica, performance-first  
- **Deploy:** static hosting

---

## 📁 Struttura del progetto
```plaintext/
├── content/        # Contenuti del sito
├── layouts/        # Template personalizzati
├── static/         # Asset statici (immagini, favicon, ecc.)
├── config.yaml     # Configurazione principale
└── README.md
```
